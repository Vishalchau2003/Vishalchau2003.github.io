About

From sensor to inference.

I work across the full stack of applied ML — from cleaning and preparing raw sensor or image data, to training and optimizing models, to squeezing them onto constrained hardware like the ESP32. My strongest interest is healthcare AI: detecting AFib from raw ECG signal, and classifying skin lesions from images, both while keeping the models small enough to run in real time.

Outside of healthcare, I build practical ML and embedded tools — a helmet-detection safety system that controls vehicle ignition, an air-quality sensor node, and accessibility tools that read and check text aloud. I'm currently expanding into front-end web development so I can ship these projects as interactive products, not just notebooks.
